Lesson 3 Reference Code

Links and Media

<!-- EXTERNAL LINKS: Full URL with https:// -->
<a href="https://www.example.com">Visit Example.com</a>

<!-- INTERNAL LINKS: Path to another page on your site -->
<a href="about.html">About Page</a>
<a href="../index.html">Back to Home</a>

<!-- ANCHOR LINKS: Jump to a section on the same page -->
<a href="#contact">Jump to Contact Section</a>
<h2 id="contact">Contact Us</h2>

<!-- EMAIL LINKS: Opens user's email client -->
<a href="mailto:hello@school.edu">Email Us</a>

<!-- OPEN IN NEW TAB: Use target attribute -->
<a href="https://example.com" target="_blank">Open in New Tab</a>

<!-- IMAGES: src is path/URL, alt describes the image -->
<img src="images/logo.png" alt="School Logo">
<img src="photo.jpg" alt="Students in classroom"
     width="300" height="200">

<!-- CLICKABLE IMAGES: Wrap img in an anchor tag -->
<a href="gallery.html">
  <img src="thumbnail.jpg" alt="View Gallery">
</a>

<!-- VIDEO: controls attribute adds play/pause buttons -->
<video src="intro.mp4" controls width="500">
  Your browser doesn't support video.
</video>

<!-- AUDIO: Same pattern as video -->
<audio src="podcast.mp3" controls>
  Your browser doesn't support audio.
</audio>

Key Concepts:

  • href attribute - Hypertext reference, the destination URL
  • Relative paths - ./file.html (same folder), ../file.html (parent folder)
  • Absolute paths - Full URL starting with https://
  • alt text - REQUIRED for images, describes content for screen readers and broken images
  • controls attribute - Adds playback controls to video/audio
