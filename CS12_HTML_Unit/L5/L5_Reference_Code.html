Lesson 5 Reference Code

HTML Forms

<form action="/submit" method="post">

  <!-- Label links to input via matching for/id -->
  <label for="username">Name:</label>
  <input type="text" id="username"
         name="username" required>

  <label for="email">Email:</label>
  <input type="email" id="email"
         name="email">

  <!-- Radio buttons share the same name -->
  <input type="radio" name="grade"
         id="g11" value="11">
  <label for="g11">Grade 11</label>

  <!-- Dropdown -->
  <select name="subject">
    <option value="cs">Computer Science</option>
    <option value="math">Math</option>
  </select>

  <label for="msg">Message:</label>
  <textarea id="msg" name="msg"
            rows="4"></textarea>

  <button type="submit">Send</button>
</form>

Common Input Types:

<!-- TEXT INPUT TYPES -->
<input type="text">       <!-- Single-line text -->
<input type="email">      <!-- Email validation -->
<input type="password">   <!-- Hidden characters -->
<input type="tel">       <!-- Phone number -->
<input type="url">       <!-- Website URL -->
<input type="number">     <!-- Numeric input -->
<input type="date">       <!-- Date picker -->

<!-- CHOICE INPUT TYPES -->
<input type="checkbox">   <!-- Multiple selections -->
<input type="radio">      <!-- Single selection from group -->

<!-- OTHER INPUT TYPES -->
<input type="file">       <!-- File upload -->
<input type="color">      <!-- Color picker -->
<input type="range">      <!-- Slider -->

Key Form Attributes:

<!-- FORM ATTRIBUTES -->
<form action="/submit"   <!-- URL to send data to -->
      method="post">    <!-- How to send: "get" or "post" -->

<!-- INPUT ATTRIBUTES -->
<input name="username"      <!-- Key for this field's data -->
       id="username"        <!-- Links to label's for attribute -->
       value="default"      <!-- Pre-filled value -->
       placeholder="hint"   <!-- Hint text inside field -->
       required             <!-- Must be filled before submit -->
       disabled             <!-- Field is grayed out -->
       readonly>            <!-- Can't be edited -->

Key Concepts:

  • <form> - Container that groups inputs; action is where data goes
  • <label> - Associates text with an input using for/id pairing
  • name attribute - The key used when form data is submitted (required!)
  • id attribute - Links label to input, used for CSS/JS targeting
  • Radio buttons - Must share the same name to be mutually exclusive
  • Checkboxes - Use different names to allow multiple selections
  • <select> - Dropdown menu containing <option> elements
  • <textarea> - Multi-line text input; set rows/cols for size
  • required - Browser validates field before allowing submission
