Lesson 4 Reference Code

Lists and Tables

<!-- UNORDERED LIST: Bullets, order doesn't matter -->
<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

<!-- ORDERED LIST: Numbers, order matters -->
<ol>
  <li>Step one</li>
  <li>Step two</li>
  <li>Step three</li>
</ol>

<!-- NESTED LISTS: Put a list inside an <li> -->
<ul>
  <li>Web Technologies
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Other Topics</li>
</ul>

<!-- BASIC TABLE STRUCTURE -->
<table>
  <tr> <!-- Table Row -->
    <th>Header 1</th> <!-- Table Header (bold/centered) -->
    <th>Header 2</th>
    <th>Header 3</th>
  </tr>
  <tr>
    <td>Data A</td> <!-- Table Data -->
    <td>Data B</td>
    <td>Data C</td>
  </tr>
  <tr>
    <td>Data D</td>
    <td>Data E</td>
    <td>Data F</td>
  </tr>
</table>

<!-- SEMANTIC TABLE: Using thead, tbody, tfoot -->
<table>
  <thead> <!-- Table header section -->
    <tr>
      <th>Name</th>
      <th>Grade</th>
      <th>Score</th>
    </tr>
  </thead>
  <tbody> <!-- Table body section -->
    <tr>
      <td>Alice</td>
      <td>11</td>
      <td>95%</td>
    </tr>
    <tr>
      <td>Bob</td>
      <td>12</td>
      <td>88%</td>
    </tr>
  </tbody>
  <tfoot> <!-- Table footer (optional) -->
    <tr>
      <td colspan="2">Average:</td>
      <td>91.5%</td>
    </tr>
  </tfoot>
</table>

<!-- SPANNING COLUMNS/ROWS -->
<table>
  <tr>
    <th colspan="2">Header spans 2 columns</th>
  </tr>
  <tr>
    <td>Cell 1</td>
    <td>Cell 2</td>
  </tr>
</table>

Key Concepts:

  • <ul> - Unordered list (bullets), use when order doesn't matter
  • <ol> - Ordered list (numbers), use for steps or rankings
  • <li> - List item, must be inside ul or ol
  • <th> - Table header cell (bold, centered by default)
  • <td> - Table data cell (normal text)
  • colspan/rowspan - Make a cell span multiple columns or rows
  • thead/tbody/tfoot - Semantic grouping for better accessibility
